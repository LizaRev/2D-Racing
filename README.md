# 🏎️ 2D Racing Game

A simple 2D racing game built with C++ and SFML.

---

## 🎮 Controls

⬆️ Up Arrow — Accelerate

⬇️ Down Arrow — Brake / Reverse

⬅️ Left Arrow — Turn Left

➡️ Right Arrow — Turn Right

---

## ✨ Features

🚗 Smooth car movement

🛣️ Road boundary detection

🚧 Random obstacle generation

💥 Collision detection

⚡ Acceleration and braking mechanics

🎯 Simple and responsive controls

---

## 🕹️ Gameplay

Drive your car along the road while avoiding obstacles.

The game ends when:

• You collide with an obstacle

• You leave the road boundaries

---

## 🛠️ Requirements

• C++11 or newer

• SFML 2.5+

• GCC / Clang / MSVC

---

## 🚀 Build

```bash
g++ main.cpp -o RacingGame \
-lsfml-graphics \
-lsfml-window \
